# NodeJS Template

Basic Template for node projects

# Project Structure

```
  ├── app			               # App content
  │   ├── models           	   # All collection schemas
  │   ├── routes           	   # All routes
  │   ├── services           	   # All function modules
  │   ├── utils            	   # all helper tools
  ├── docs                       # Documentation files
  │   ├── api_collection         # dump of latest postman collection
	│   ├── flow_diagrams          # All flow diagrams of modules schemas
	├── scripts                    # all bash scripts
	├── config                     # all config variables are stored here
	├── bin                        # Http server setup
	├── app.js                     # Main server interface
	├── package.json               # project dependices and meta info
	├── .env                       # create this file and define all environment variables
	├── .gitignore                 # List files/folders to untrack in git
```
